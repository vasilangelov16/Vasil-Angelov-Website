## Songs needing lyrics (3)

Edit `server/data/songs.json`. Add lyrics in this format:

```json
"lyrics":"[Verse 1]\nLine one\nLine two\n\n[Chorus]\nChorus line\n\n[Verse 2]\n..."
```

- Use `\n` for newlines
- Use `[Verse 1]`, `[Chorus]`, etc. for section labels (they get styled in the app)
- When a chorus repeats word-for-word consecutively, use `[Chorus xN]` (e.g. `[Chorus x2]`) instead of repeating the text
